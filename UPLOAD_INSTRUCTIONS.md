# Upload Instructions

## ChatGPT Project upload

1. Open ChatGPT.
2. Create a new Project.
3. Upload this zip file:

`aurora-edge-group-chatgpt-upload.zip`

4. Also upload or paste:

`CHATGPT_HANDOFF.md`

5. Paste the prompt from `CHATGPT_HANDOFF.md` into the first message.

## Recommended upload contents

If you do not want to upload the full folder, upload these first:

- `CHATGPT_HANDOFF.md`
- `docs/`
- `site/`
- `scripts/`
- `campaigns/`
- `templates/`
- `aws/`

## Important note

This project includes AWS resource names, public site URLs, and SES case info.
Review before sharing outside your own ChatGPT account.
